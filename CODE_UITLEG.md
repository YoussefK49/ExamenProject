# Code Uitleg voor Leraren

## PHP (Server-Side Scripting)

PHP is een server-side scripttaal die wordt gebruikt voor dynamische webontwikkeling. In dit project wordt PHP gebruikt om de backend-logica te verwerken, zoals gebruikersauthenticatie, database-interacties, en het genereren van HTML-content op basis van gebruikersgegevens. PHP-code wordt op de server uitgevoerd voordat de HTML naar de browser wordt gestuurd, wat betekent dat gebruikers de broncode niet kunnen zien. Het project gebruikt PHP sessies om gebruikersingelogd te houden en $_POST en $_GET superglobals om formuliergegevens te verwerken. De db.php file bevat database-functies die gebruikmaken van MySQLi prepared statements om SQL-injectie te voorkomen. De auth.php file bevat authenticatie-functies zoals isLoggedIn(), requireLogin(), en getCurrentUserId() die de sessie controleren. PHP wordt ook gebruikt om conditionele HTML te genereren, zoals het tonen van verschillende content voor ingelogde en uitgelogde gebruikers. Het verwerkt acties zoals liken, commentaren toevoegen, en posts maken door POST-requests te ontvangen en de juiste functies aan te roepen. Na het verwerken van een actie wordt de gebruiker vaak omgeleid met header('Location: ...') om page refresh te voorkomen. PHP is essentieel voor de veiligheid en functionaliteit van de applicatie.

## CSS (Cascading Style Sheets)

CSS wordt gebruikt voor de visuele styling en layout van de webpagina's. In dit project definieert styles.css hoe alle HTML-elementen eruitzien en hoe ze gepositioneerd zijn. CSS gebruikt variabelen (custom properties) in de :root selector om consistente kleuren, schaduwen, en andere design tokens te definiëren die door de hele applicatie worden gebruikt. Het project ondersteunt zowel een light als dark thema via de [data-theme="dark"] selector die de CSS-variabelen overschrijft. CSS wordt gebruikt voor responsive design met media queries die de layout aanpassen voor verschillende schermformaten. De navbar is sticky positioned zodat hij bovenaan blijft bij scrollen. Post cards hebben hover effects met box-shadow en transform voor een interactieve ervaring. Modals hebben een animatie met @keyframes voor een smooth slide-in effect. CSS Grid en Flexbox worden gebruikt voor layout management, zoals het centreren van content en het alignen van items in de navbar. Border-radius wordt gebruikt voor afgeronde hoeken, en gradients voor avatar achtergronden. CSS transitions zorgen voor smooth hover effects op knoppen en links. De styling is modern met subtiele schaduwen, goede spacing, en een consistent kleurenpalet. CSS zorgt ervoor dat de applicatie er professioneel en gebruiksvriendelijk uitziet.

## JavaScript (Client-Side Scripting)

JavaScript wordt gebruikt voor interactieve functionaliteit aan de client-side. In dit project beheert script.js de thema-wisselaar die tussen light en dark modus schakelt. Het controleert localStorage om de gebruikersvoorkeur te onthouden en past deze toe bij het laden van de pagina. Als er geen voorkeur is opgeslagen, wordt de systeemvoorkeur (prefers-color-scheme) gebruikt. JavaScript voegt event listeners toe aan elementen zoals de themeToggle knop om te reageren op gebruikersinteracties. Het beheert ook de mobiele menu toggle en dropdown menu's voor de navigatie. JavaScript wordt gebruikt voor DOM manipulatie, zoals het tonen en verbergen van modals door de style.display property te wijzigen. De window.onclick event handler zorgt ervoor dat modals sluiten wanneer er buiten de modal wordt geklikt. JavaScript kan ook gebruikt worden voor AJAX requests om data asynchroon op te halen zonder de pagina te refreshen, hoewel dit project voornamelijk traditionele form submissions gebruikt. JavaScript draait in de browser en kan direct de DOM manipuleren, wat zorgt voor een snelle en responsieve gebruikerservaring. Het is essentieel voor moderne webapplicaties om interactief te zijn zonder constant naar de server te communiceren voor elke kleine verandering.

## SQL (Structured Query Language)

SQL wordt gebruikt voor database-operaties en data-management. In dit project bevat database.sql het schema voor de MySQL database met tabellen voor users, posts, comments, likes, en follows. SQL CREATE TABLE statements definiëren de structuur van elke tabel met kolommen, data types, en constraints zoals PRIMARY KEY, FOREIGN KEY, en UNIQUE. De users tabel heeft kolommen voor id, username, email, password_hash, avatar, bio, en timestamps. De posts tabel slaat post content op met een foreign key naar de users tabel. SQL SELECT queries worden in PHP gebruikt om data op te halen, zoals getPosts() die posts met gebruikersinformatie ophaalt via JOINs. INSERT statements worden gebruikt om nieuwe data toe te voegen, zoals bij createPost() en addComment(). UPDATE statements wijzigen bestaande data, zoals updateBio() die de bio kolom van een gebruiker bijwerkt. DELETE statements verwijderen data, zoals bij unfollowUser(). SQL WHERE clauses filteren resultaten op basis van voorwaarden. Prepared statements met parameter binding worden gebruikt om SQL-injectie te voorkomen door user input veilig te verwerken. SQL is de taal waarmee de applicatie communiceert met de database om alle gebruikersdata, posts, en interacties op te slaan en op te halen. Zonder SQL zou de applicatie geen permanente data kunnen opslaan.
